# GETUM
[포트폴리오 페이지 : getum.cco.kr](http://getum.cco.kr)   
GETUM은 텀블러를 판매하는 가상 쇼핑몰입니다.   
Spring Framework을 사용하지 않은 웹프로젝트입니다.   

### 개발기간
2020년 3월 2일 ~ 4월9일 (일부 기능 미구현)

### 서버    
* Raspberry pi 4
* OS: Raspbian GNU/Linux 10 (buster)

### 사용 기술
* JAVA 8
* MariaDB 10.3.22
* TOMCAT 8.0
* HTML5, CSS3, JS, JQuery, Bootstrap4
